How to Use Polyglot Bot

Adding to Your Server

Click the "Add to Discord" button above and authorize the bot to access your server with translation permissions.

Basic Translation

Type !translate [text] in any channel and the bot will translate it to the default language.

Auto-Translation

Use !autotranslate on in a channel to enable automatic translation of all messages.

Language Selection

Change the target language with !setlang [language code] (e.g., !setlang es for Spanish).
